# 🌐 HTML & CSS Practice Projects

This repository contains beginner-friendly examples and exercises using **HTML** and **CSS**. These mini-projects are great for learning the fundamentals of **web page structure**, **styling**, and **responsive design**.

---

## 📚 What’s Inside

- 📄 Semantic HTML structure
- 🎨 CSS styling and layout techniques
- 🧱 Flexbox and grid examples
- 📱 Responsive design basics
- 🎯 Practice templates and mini websites

---

## 🚀 Getting Started

1. Clone this repo:

```bash
git clone https://github.com/DoNguyenAnhTuan/HTML-CSS.git
````

2. Open any `.html` file in your browser to preview.

3. Modify the HTML/CSS files and experiment!

---

## 🎓 Who is this for?

* Students starting with **web development**
* Educators creating demo content
* Anyone wanting to improve their frontend layout skills
